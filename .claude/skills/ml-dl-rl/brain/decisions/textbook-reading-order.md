# Decision: Suggested Reading Order Across the 11 Books

A pragmatic path through the brain depending on what you're trying to learn.

## Foundations First (~6–8 months self-study)
1. [[../summaries/probabilistic-ml-introduction]] — Murphy intro, build broad ML literacy
2. [[../summaries/algorithms-for-optimization]] — optimization toolkit
3. [[../summaries/understanding-deep-learning]] — modern DL principles

## Then RL Track
4. [[../summaries/reinforcement-learning-introduction]] — Sutton & Barto, foundational
5. [[../summaries/algorithms-for-decision-making]] — broader decision-theoretic view
6. [[../summaries/multi-agent-reinforcement-learning]] — multi-agent extension

## Or Productionization Track
4'. [[../summaries/machine-learning-systems]] — engineering ML in production

## Advanced / Reference (Year 2+)
7. [[../summaries/probabilistic-ml-advanced]] — deep Bayesian ML reference
8. [[../summaries/decision-making-under-uncertainty]] — applications-deep companion to (5)

## Specialized
9. [[../summaries/fairness-and-machine-learning]] — when working on consequential decisions
10. [[../summaries/algorithms-for-validation]] — when shipping safety-critical autonomy

See [[stanford-algorithms-trilogy]] for the Kochenderfer through-line and how these books interlock.
