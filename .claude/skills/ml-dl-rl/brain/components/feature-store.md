# Feature Store

Centralized service for ML features: ingests raw data, computes/transforms features, serves them with low latency at inference and consistently for offline training.

Solves training-serving skew: same feature pipeline for both. Examples: Feast, Tecton, Vertex AI Feature Store, SageMaker Feature Store.
