# No-Regret Learning

Online learning algorithms whose average regret → 0 as T → ∞. In games, no-regret learners converge to coarse correlated equilibria.

## Examples
- Multiplicative weights (Hedge / EXP3)
- Follow-the-regularized-leader
- Counterfactual regret minimization (CFR) — solved heads-up no-limit poker (Libratus)
