# Probabilistic Machine Learning: An Introduction (Murphy 2022)

A comprehensive, modern introduction to ML grounded in probability theory, covering foundations through deep learning to unsupervised methods.

## Identification
- **Author:** Kevin P. Murphy (Google Research)
- **Publisher:** MIT Press, 2022
- **License:** CC-BY-NC-ND
- **Series:** Adaptive Computation and Machine Learning
- **Source:** [[../entities/book-pml1]]

## Structure

### Part I — Foundations (Ch. 2–8)
- Probability: univariate ([[../concepts/probability-distributions]]) and multivariate models
- Statistics: [[../concepts/maximum-likelihood-estimation]], MAP, [[../concepts/bayesian-inference]], regularization
- [[../concepts/decision-theory]]: Bayes risk, ROC curves, A/B testing
- [[../concepts/information-theory]]: entropy, KL, mutual information
- Linear algebra and [[../concepts/optimization]] (first/second-order, SGD, constrained)

### Part II — Linear Models (Ch. 9–12)
- Linear discriminant analysis, logistic regression, linear regression
- Generalized linear models

### Part III — Deep Neural Networks (Ch. 13–15)
- [[../components/mlp-tabular]] for tabular data
- [[../components/cnn]] for images
- [[../components/rnn-sequences]] for sequences

### Part IV — Nonparametric Models (Ch. 16–18)
- Exemplar-based methods (kNN)
- Kernel methods, SVMs, Gaussian processes
- Trees, random forests, [[../concepts/boosting]]

### Part V — Beyond Supervised (Ch. 19–23)
- Semi-supervised, self-supervised, active learning
- Dimensionality reduction (PCA, autoencoders)
- Clustering, recommender systems, graph embeddings

## Why It Matters
- Pairs with [[probabilistic-ml-advanced]] (vol. 2) — same author, deeper material
- Most current "Bishop-style" probabilistic text; widely used as a graduate ML reference
- Bridges classical statistical ML and modern deep learning
